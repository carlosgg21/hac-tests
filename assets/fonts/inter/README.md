# Fuentes Inter - Instrucciones de Descarga

## Estado Actual
El archivo `fonts.css` está configurado y listo para usar. Sin embargo, los archivos de fuente WOFF2 aún no se han descargado.

## Cómo Descargar las Fuentes

### Opción 1: Google Webfonts Helper (Recomendado)
1. Visita: https://gwfh.mranftl.com/fonts/inter?subsets=latin
2. Selecciona los siguientes pesos: **300**, **400**, **500**, **600**, **700**, **800**
3. Selecciona "Modern Browsers" para obtener solo WOFF2
4. Haz clic en "Download @font-face kit"
5. Extrae los archivos `.woff2` y colócalos en este directorio (`assets/fonts/inter/`)
6. Renombra los archivos según estos nombres:
   - `inter-v18-latin-300.woff2`
   - `inter-v18-latin-regular.woff2` (para weight 400)
   - `inter-v18-latin-500.woff2`
   - `inter-v18-latin-600.woff2`
   - `inter-v18-latin-700.woff2`
   - `inter-v18-latin-800.woff2`

### Opción 2: Descarga Manual desde Google Fonts
1. Visita: https://fonts.google.com/specimen/Inter
2. Selecciona los pesos necesarios
3. Descarga el archivo ZIP
4. Extrae los archivos `.woff2` del subdirectorio `static/`
5. Colócalos en este directorio con los nombres indicados arriba

### Verificación
Una vez descargados, deberías tener estos archivos en `assets/fonts/inter/`:
- fonts.css
- inter-v18-latin-300.woff2
- inter-v18-latin-regular.woff2
- inter-v18-latin-500.woff2
- inter-v18-latin-600.woff2
- inter-v18-latin-700.woff2
- inter-v18-latin-800.woff2
